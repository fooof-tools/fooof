    fooof.utils.reports.methods_report_info¶

    fooof.utils.reports.methods_report_info(fooof_obj=None, concise=False)[source]¶

    Prints out a report of information required for methods reporting.

    Parameters
    fooof_objFOOOF or FOOOFGroup, optional

    An object with setting information available. If provided, is used to collect and print information to be reported.

    concisebool, optional, default: False

    Whether to print the report in concise mode.

    Notes

    Any missing values (information that is not available) will be printed out as ‘XX’.
