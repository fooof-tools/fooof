    fooof.utils.reports.methods_report_text¶

    fooof.utils.reports.methods_report_text(fooof_obj=None)[source]¶

    Prints out a text template of methods reporting information.

    Parameters
    fooof_objFOOOF or FOOOFGroup, optional

    An object with setting information available. If None, the text is returned as a template, without values.

    Notes

    Any missing values (information that is not available) will be printed out as ‘XX’.
